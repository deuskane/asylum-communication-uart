# UART
CSR for UART

| Address | Registers |
|---------|-----------|
|0x0|data|
|0x1|ctrl|

## 0x0 data
Write : data to tansmit, Read : data to receive

### [7:0] value
Data with data_oe with mask apply

## 0x1 ctrl
Write : data to tansmit, Read : data to receive

### [7:0] value
Data with data_oe with mask apply
